## hệ thống: Shopee (ứng dụng thương mại điện tử)

| **Nhóm yếu tố**                                               | **Yếu tố cụ thể**                                          | **Mô tả vai trò / ảnh hưởng đến hệ thống**                                |
| ------------------------------------------------------------- | ---------------------------------------------------------- | ------------------------------------------------------------------------- |
| **1. Người dùng (Users)**                                     | - Người mua (khách hàng)                                   | Sử dụng ứng dụng để tìm kiếm, đặt hàng, thanh toán, đánh giá sản phẩm.    |
|                                                               | - Người bán (shop, doanh nghiệp)                           | Đăng sản phẩm, xử lý đơn hàng, quản lý kho và theo dõi doanh thu.         |
|                                                               | - Nhân viên Shopee (quản trị, CSKH, kiểm duyệt)            | Quản lý hệ thống, hỗ trợ khách hàng, xử lý khiếu nại, kiểm soát nội dung. |
| **2. Phần cứng (Hardware)**                                   | - Máy chủ trung tâm (server)                               | Lưu trữ dữ liệu người dùng, sản phẩm, đơn hàng, giao dịch.                |
|                                                               | - Thiết bị người dùng (điện thoại, máy tính bảng, PC)      | Chạy ứng dụng Shopee hoặc truy cập qua trình duyệt web.                   |
|                                                               | - Hạ tầng mạng, CDN                                        | Hỗ trợ tốc độ truy cập nhanh, phân phối nội dung toàn cầu.                |
| **3. Phần mềm (Software)**                                    | - Ứng dụng Shopee (Android, iOS, Web)                      | Giao diện người dùng chính, nơi diễn ra mọi giao dịch.                    |
|                                                               | - Hệ quản trị cơ sở dữ liệu (Database Management System)   | Lưu trữ dữ liệu người dùng, sản phẩm, đơn hàng.                           |
|                                                               | - Hệ thống phân tích dữ liệu (Data Analytics, AI/ML)       | Đề xuất sản phẩm, cá nhân hóa trải nghiệm mua sắm.                        |
|                                                               | - Phần mềm quản lý vận hành nội bộ                         | Dùng cho CSKH, kiểm duyệt sản phẩm, xử lý gian lận.                       |
| **4. Hệ thống bên ngoài (External Systems)**                  | - Cổng thanh toán (Momo, ZaloPay, Visa, Mastercard…)       | Xử lý thanh toán trực tuyến cho đơn hàng.                                 |
|                                                               | - Hệ thống vận chuyển (GHN, GHTK, Shopee Express, VNPost…) | Kết nối để lấy đơn, cập nhật trạng thái vận chuyển.                       |
|                                                               | - Mạng xã hội (Facebook, TikTok, Instagram)                | Kênh marketing, quảng cáo sản phẩm.                                       |
|                                                               | - Cơ quan thuế, ngân hàng                                  | Liên quan đến thanh toán, đối soát, và nghĩa vụ thuế.                     |
| **5. Quy trình nghiệp vụ (Business Processes)**               | - Đăng ký và xác thực tài khoản                            | Người dùng tạo tài khoản, xác minh email/số điện thoại.                   |
|                                                               | - Đăng bán và mua hàng                                     | Người bán đăng sản phẩm → người mua đặt hàng → thanh toán → vận chuyển.   |
|                                                               | - Xử lý đơn hàng, hoàn tiền, khiếu nại                     | Theo dõi tình trạng đơn, xử lý sự cố, chăm sóc khách hàng.                |
| **6. Luật lệ và chính sách (Legal & Regulatory Environment)** | - Luật thương mại điện tử Việt Nam                         | Quy định về giao dịch, bảo vệ người tiêu dùng, trách nhiệm sàn TMĐT.      |
|                                                               | - Luật bảo vệ dữ liệu cá nhân (Nghị định 13/2023/NĐ-CP)    | Bảo mật thông tin người dùng, hạn chế rò rỉ dữ liệu.                      |
|                                                               | - Chính sách thuế và kế toán điện tử                       | Áp dụng với người bán kinh doanh trên sàn.                                |
|                                                               | - Quy định của Ngân hàng Nhà nước                          | Liên quan đến thanh toán không dùng tiền mặt.                             |
