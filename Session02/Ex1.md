## Hệ thống quản lý học trực tuyến

| **Nhóm Vai Trò**                               | **Stakeholder Cụ thể**                               | **Mô tả vai trò / Mối quan tâm chính**                                                                                                             |
| ---------------------------------------------- | ---------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. Người dùng cuối (End Users)**             | - Sinh viên / học viên                               | Truy cập khóa học, làm bài tập, xem điểm, tương tác với giảng viên. Quan tâm đến tính dễ dùng, ổn định và trải nghiệm học tập.                     |
|                                                | - Giảng viên                                         | Tạo khóa học, quản lý nội dung, đánh giá học viên, theo dõi tiến độ. Quan tâm đến công cụ hỗ trợ giảng dạy, báo cáo, và tính năng quản lý lớp học. |
|                                                | - Quản lý đào tạo / Điều phối viên                   | Quản lý chương trình, phân công giảng viên, theo dõi tiến độ học tập. Quan tâm đến dữ liệu tổng hợp, báo cáo, và khả năng giám sát.                |
|                                                | - Phụ huynh (nếu hệ thống có chức năng theo dõi)     | Theo dõi kết quả học tập của con, quan tâm đến tiến độ và thành tích học tập.                                                                      |
| **2. Sponsor (Nhà tài trợ / Chủ đầu tư)**      | - Ban giám hiệu / Lãnh đạo tổ chức đào tạo           | Cung cấp ngân sách, phê duyệt dự án, xác định mục tiêu và phạm vi. Quan tâm đến hiệu quả đầu tư, tính minh bạch, và ROI.                           |
|                                                | - Cơ quan quản lý giáo dục (nếu là dự án công)       | Đặt yêu cầu, giám sát tiến độ, kiểm định chất lượng.                                                                                               |
| **3. Chuyên gia nghiệp vụ (Business Experts)** | - Chuyên viên đào tạo / giáo vụ                      | Hiểu rõ quy trình học tập, quy chế đào tạo, đưa ra yêu cầu chức năng phù hợp.                                                                      |
|                                                | - Chuyên gia thiết kế chương trình học               | Đưa ra tiêu chuẩn nội dung, cấu trúc khóa học, đề xuất cải tiến nghiệp vụ.                                                                         |
| **4. Bộ phận kỹ thuật (Technical Team)**       | - Quản trị hệ thống (System Admin)                   | Cài đặt, cấu hình, bảo trì hệ thống, sao lưu dữ liệu.                                                                                              |
|                                                | - Nhóm phát triển phần mềm                           | Phân tích yêu cầu, thiết kế, lập trình, kiểm thử, triển khai hệ thống.                                                                             |
|                                                | - Nhóm hỗ trợ kỹ thuật (Helpdesk)                    | Giải quyết sự cố, hỗ trợ người dùng.                                                                                                               |
|                                                | - Quản trị cơ sở dữ liệu (DBA)                       | Quản lý dữ liệu, đảm bảo an toàn và hiệu năng.                                                                                                     |
| **5. Bên thứ ba (Third Parties)**              | - Nhà cung cấp hạ tầng (Cloud, Hosting)              | Cung cấp máy chủ, dịch vụ lưu trữ, CDN, bảo mật.                                                                                                   |
|                                                | - Đơn vị tích hợp thanh toán (ví điện tử, ngân hàng) | Hỗ trợ thanh toán học phí trực tuyến.                                                                                                              |
|                                                | - Nhà cung cấp nội dung học liệu (Content Providers) | Cung cấp tài nguyên học tập, video, tài liệu số.                                                                                                   |
|                                                | - Đối tác chứng chỉ / kiểm định                      | Cung cấp chứng nhận hoặc đánh giá chất lượng khóa học.                                                                                             |
